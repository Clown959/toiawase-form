# toiawase-form

## 環境構築
1.docker-compose exec php bash
2.composer install
3..env.exampleファイルから.envを作成し、環境変数の変更
4.php artisan key:generate
php artisan migrate
php artisan db:seeg


## 使用技術(実行環境)
・PHP8.0
・Laravel 10.0
・MySQL 8.0
